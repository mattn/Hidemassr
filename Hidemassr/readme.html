ダウンロード

  • hidematter02.lzh　Version 0.2　(2010/04/19)

概要

  • 秀丸エディタから Twitter へ投稿します。
  • 長い文章を 1 行ずつ or 140 字ごとに自動分割して連続投稿できます。
  • j.mp による URL 短縮機能が統合されています。
  • 今のところ書き込み専用です。表示機能はありません。

セットアップ

インストール

  1. 秀丸エディタ ver 8.00 以降*1と Windows 98 以降*2をお使いですね？
  2. マクロフォルダの位置を確認します。マクロフォルダは秀丸の [その他]-[動作環境]-[パス]-[環境]-[マクロファイル用のフォルダ] で指定されているフォルダです。無指定なら Hidemaru.exe と同じフォルダです。
  3. ダウンロードしたアーカイブをマクロフォルダに解凍します。
  4. [マクロ]-[マクロ登録...] で Hidematter.mac を適当な番号に登録します。
  5. [その他]-[キー割り当て...] で適当なキーに割り当てます。

インストール後のファイル構成は次のようになります。INI ファイルは初回起動時に自動生成されます。

■マクロフォルダ\
│・Hidematter.mac
└■Hidematter\
　　・hidematr.mac
　　・wshTweet.js
　　・shortenUrls.js
　　・user=Windowsユーザ名.ini

アップデート

  1. 新しいアーカイブをマクロフォルダに上書き解凍します。

アンインストール

  1. インストール時と同じ方法でキー割り当てを解除します。
  2. 同様にマクロ登録も解除します。
  3. Hidematter.mac と Hidematter フォルダを削除します。レジストリは使っていません。
*1
秀丸エディタ v7.x 以前では Hidematter は使えません。
*2
厳密には IE4 以降…で動くと思うけど、動作不確認。現実的には IE6 以降を推奨。

使い方と動作

  1. 投稿したい文章を範囲選択します。無選択時はカーソルのある行全体が投稿されます。
  2. Hidematter.mac を実行します。
  3. 初回起動時は Twitter のユーザー名とパスワードを入力します。
  4. 投稿が行われます。
    • 成功した場合、投稿された文章に下線が引かれます。
    • 認証に失敗した場合、メッセージが表示され、記憶されていたユーザー名とパスワードがリセットされます。
    • その他の原因で失敗した場合、理由を示すメッセージが表示されます。
  • 複数行選択した場合、1 行ずつ別々に投稿するか、改行も含めてそのまま投稿するかを選べます。
  • 投稿範囲が 140 字を超える場合、自動的に分割されて連続投稿されます。

ユーザー名とパスワードについて

  • ユーザー名はファイルに保存され、再起動後も有効となります。
  • パスワードは共有メモリに記憶され、すべての秀丸を終了するまで有効となります。
  • パスワードは平文で記憶・送信されます。他の (悪意ある) 秀丸マクロは共有メモリに記憶されているパスワードを読み取ることが可能です。
  • 保存されているユーザー名を変更したい場合は、パスワード入力ダイアログで [キャンセル] を押してください。

お約束

動作環境

  • 秀丸エディタ v8.00 以降 + Internet Explorer 4 以降で動作するはずです。
  • 秀丸エディタ v8.01β1 + Internet Explorer 8 で動作確認しました。
  • マクロ内部で Web サービスを利用しているため、インターネット接続がないと動作しません。

既知の問題

  • パスワードは ASCII 文字に限られる。
  • パスワードに「"」が含まれていると認証できないと思われる。
  • 現在使っている Basic 認証は 2010 年 6 月いっぱい? で使えなくなる見込み。それまでに oAuth か xAuth へ移行する必要がある。
